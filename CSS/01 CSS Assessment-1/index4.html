What is CSS?

  1. CSS stands for Cascading Style Sheet.

  2. CSS tells us how HTML elements are to be displayed on screen, or on paper.

  3. CSS is not necessary CSS is saves a lot of work. after this there is not much to explain.

  4. External style sheet are stored in another CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline Elements- used to style the elements in the HTML tag itself.
  • Internal Elements- used to style the elements, it can be done in the HEAD tag.
  • External Elements- used to style the elements by linking to another CSS page.
